# ArtOfRallyLeaderboardAPI
Documentation of the Art of Rally leaderboard API

# Get Leaderboard

```
https://www.funselektorfun.com/artofrally/leaderboard/
  [:leaderboard-string]/
  [:filter-id]/
  [:platform]/
  [:user-id]/
  [:friends]
```

Friends are formatted as `[:userid-1,:userid-2,...]` **including**
the square brackets.
